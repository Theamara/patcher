WeatherColorQuad (Skyrim)
Summary
Represents a set of weather colors for each of the four parts of the day: sunrise, day, sunset and night.
Properties
Name Description
Color Day Gets or sets the weather color during the day.
Color Night Gets or sets the weather color during the night.
Color Sunrise Gets or sets the weather color during the sunrise.
Color Sunset Gets or sets the weather color during the sunset.
