Effect (Skyrim)
Summary
Represents a custom effect based on a Magic Effect.
Properties
Name Description
int Area Gets or sets the radius of the area covered with this Effect.
Mgef BaseEffect Gets or sets the base Magic Effect of this Effect.
ConditionCollection Conditions Gets or sets the ConditionCollection contaning Conditions that detemine when this Effect is active.
int Duration Gets or sets the duration of this Effect.
float Magnitude Gets or sets the magnitude of this Effect.
Remarks
New effects can be created using the Engine helper method Engine.CreateEffect(). When a new custom effect has been created, conditions can be added to it, if desired. And finally, the new effect can be added to an EffectCollection associated with the Target form.
