ProjectileTypes (Skyrim)
Summary
Defines Projectile types.
Fields
Name Description
Arrow Indicates the Projectile is an arrow.
Barrier Indicates the Projectile is a barrier.
Beam Indicates the Projectile is a beam.
Cone Indicates the Projectile has the form of a cone.
Flame Indicates the Projectile is a flame.
Lobber Indicates the Projectile is a thrown object with a high arc.
Missile Indicates the Projectile is a missile going in a straigt line.
