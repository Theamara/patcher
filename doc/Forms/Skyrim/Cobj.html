Summary
Represents a Constructible Object form.
Properties
Name Description

uint FormId Gets the Form ID associated with this form.

Form Result Gets or sets the Form that represents the item that will be produced by this Constructible Object.
int ResultCount Gets or sets the number of items this Constructible Object will produce.
Kywd Workbench Gets or sets the Keyword that represents the crafting station or bench where this Constructible Object can be used.
