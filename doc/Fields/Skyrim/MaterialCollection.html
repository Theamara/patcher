MaterialCollection
Summary
Represents a collection of Materials.
Properties
Name Description
int Count Gets the number of materials in this collection.
Methods
Name Description
void Add(Form,int) Creates a new Material based on the specified form item and count and adds it to this collection.
void Add(Material) Adds an existing Material to this collection.
void Clear() Removes all Materials from this collection.
void Remove(Material) Removes a Material from this collection.
Remarks
Materials in a collection can be edited or removed within a foreach iteration.
Return to: Index
