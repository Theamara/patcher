Alch (Skyrim)
Summary
Represents a Potion form.
Properties
Name Description
string Description Gets or sets the description of this Potion that will override the enchantment description if provided.
string EditorId Gets or sets the Editor ID associated with this form.
EffectCollection Effects Gets or sets the EffectCollection contaning Effects of this Potion.
uint FormId Gets the Form ID associated with this form.
string FullName Gets or sets the in-game name of this Potion.
FormCollection<Kywd> Keywords Gets or sets the FormCollection containing Keywords associated with this Potion.
ObjectBounds ObjectBounds Gets or sets the ObjectBounds of this Potion.
Sndr PickUpSound Gets or sets the Sound that plays when this Potion is picked up.
Sndr PutDownSound Gets or sets the Sound that plays when this Potion is put down.
PotionTypes Type Gets or sets the type of this Potion.
Sndr UseSound Gets or sets the Sound that plays when this Potion is used.
int Value Gets or sets the value of this Potion.
float Weight Gets or sets the weight of this Potion.
string WorldModel Gets or sets the path to the world model (.nif file) that will be used for this Potion.
Methods
Name Description
bool HasTag(string) Determines whether this form has been tagged with the specified text.
void Tag(string) Tags this form with the specified text.
Remarks
Potion forms include drinks and food as well.
