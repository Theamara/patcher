Ammo (Skyrim)
Summary
Represents an Ammo form.
Properties
Name Description
float Damage Gets or sets the damage caused by this Ammo.
string Description Gets or sets the description of this Ammo.
string EditorId Gets or sets the Editor ID associated with this form.
uint FormId Gets the Form ID associated with this form.
string FullName Gets or sets the in-game name of this Ammo.
bool IgnoresNormalWeaponResistance Gets or sets the value that indicates whether this Ammo ignores normal weapon damage resistance.
string InventoryImage Gets or sets the path to the inventory image (.dds file) that will be used for this Ammo.
bool IsBolt Gets or sets the value that indicates whether this Ammo is a bolt.
bool IsPlayable Gets or sets the value that indicates whether this Ammo can be used by the player character.
FormCollection<Kywd> Keywords Gets or sets the FormCollection containing Keywords associated with this Ammo.
string MessageIcon Gets or sets the path to the message icon (.dds file) that will be used for this Ammo.
ObjectBounds ObjectBounds Gets or sets the ObjectBounds of this Ammo.
Sndr PickUpSound Gets or sets the Sound that plays when this Ammo is picked up.
Proj Projectile Gets or sets the Projectile that is launched when this Ammo is used.
Sndr PutDownSound Gets or sets the Sound that plays when this Ammo is put down.
string ShortName Gets or sets the short name of this Ammo.
int Value Gets or sets the value of this Ammo.
string WorldModel Gets or sets the path to the world model (.nif file) that will be used for this Ammo.
