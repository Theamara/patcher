ConditionCollection (Skyrim)
Summary
Represents a collection of Conditions.
Properties
Name Description
int Count Gets the number of Conditions in this collection.
Methods
Name Description
void Add(Condition) Adds a Condition to this collection.
void Clear() Removes all Conditions from this collection.
void Remove(Condition) Removes a Condition from this collection.
Remarks
Use any of the Functions helper methods to create new conditions. Conditions can be removed from a collection using the Remove() method during a foreach iteration.
