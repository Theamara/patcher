Condition (Skyrim)
Summary
Represents a function with parameters, operand, operator used to compare the result of the function with the specified operand and information on which object the function will be run.

Remarks

When a condition is created with any of the Functions helper methods its default state is as if the following methods where used: RunOnSubject(), IsEqualTo(0) and AndNext(). Calling these methods on a new conditions will have no effect however these methods can still be used to modify an existing conditions.

Method RunOnPlayer() is equivalent to RunOn(Forms.Find(0x14)) however many functions will run on the player reference by default as the player is often the subject.

Methods can be used in any order and when multiple methods that change the same aspect are called the changes made by the last method will be kept.

Methods can be chained together into a single statement in a fluent interface fashion.
