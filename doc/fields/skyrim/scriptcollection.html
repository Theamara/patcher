ScriptCollection (Skyrim)
Summary
Represents a collection of Scripts attached to an object.
Properties
Name Description
int Count Gets the number of Scripts in this collection.
Script this[string] Gets the Script which has the specified name from this collection or returns null if no such script could be found.
Methods
Name Description
void Add(Script) Adds an existing Script to this collection.
void Add(string) Adds a new Script with specified name to this collection.
void Clear() Removes all Scripts from this collection.
bool Contains(string) Determines whether a Script with specified name exists in this collection.
void Remove(string) Removes the Script with the specified name from this collection.
Remarks

A script cannot appear twice in a single script collection.

Property this[string] represents an indexer that can be used to retrieve a Script with a specific name.

Return to: Index
