EffectCollection (Skyrim)
Summary
Represents a collection of Effects that can be used to specify effects of items such as Potions or Scrolls.
Properties
Name Description
int Count Gets the number of effects in this collection.
Methods
Name Description
void Add(Effect) Adds an Effect to this collection.
void Clear() Removes all Effects from this collection.
void Remove(Effect) Removes an Effect from this collection.
Remarks
Use the Engine helper method Engine.CreateEffect() to create new effects.
